🏎️ OutRun Style — Playable Ad

Ретро-гоночная мини-игра в стиле OutRun для playable ads. Управляйте машиной, собирайте звёзды и уворачивайтесь от трафика!

## 🎮 Геймплей

- **Цель**: Собрать как можно больше звёзд ⭐ за 60 секунд
- **Управление**: Нажимайте на левую/правую половину экрана для перемещения
- **Столкновение**: При аварии игра заканчивается, уцелевшие машины уезжают к горизонту
- **Победа**: Продержитесь 60 секунд без аварий

## 🚀 Запуск

1. Откройте `index.html` в браузере
2. Нажмите на экран для начала игры
3. Музыка включится автоматически при первом касании

## ⚙️ Особенности

- **Адаптивный дизайн**: Игра подстраивается под любой размер экрана
- **Ретро-стиль**: Пиксельная графика в духе OutRun
- **4 типа машин трафика**: Случайный выбор при появлении
- **Анимация аварии**: Частицы, заморозка разметки, машины уезжают к горизонту
- **Фоновая музыка**: Автоматическое воспроизведение music.mp3

## 🔧 Настройка

В коде можно изменить:

| Параметр | Где искать | Описание |
|----------|------------|----------|
| `gameDuration` | `const gameDuration = 60000;` | Длительность игры (мс) |
| `objectCount` | `const objectCount = 10;` | Количество объектов на дороге |
| `Math.random() < 0.6` | В `drawRoad()` | Шанс появления машины (0.6 = 60%) |
| `width/2 + 32.5` и `65` | В `update()` | Размер хитбокса |
| `backgroundMusic.volume` | `backgroundMusic.volume = 0.5;` | Громкость музыки (0-1) |
| `obj.y -= 5` | В блоке `collisionHappened` | Скорость отдаления машин после аварии |

- Firefox
- Safari (iOS)
- Мобильные браузеры

## 📝 Примечания

- Для работы нужны все изображения из списка выше
- Формат музыки: MP3
- Изображения: PNG (с прозрачностью для машин и пальм)
- Браузеры могут блокировать автовоспроизведение звука — музыка запускается при первом касании

## 🎨 Стиль

Игра выполнена в эстетике OutRun:
- Неоновые цвета (циан, пурпурный)
- Сетка горизонта с градиентом заката
- Пальмы по краям дороги
- Жёлтая дорожная разметка
